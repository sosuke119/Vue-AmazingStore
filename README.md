# VueSlack

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

透過線上課程的教學，所製作而成的 Vue-Slack

### 功能
  - 使用Firebase APi
  - 使用Google 登入系統

### 技能
  - Html - pug
  - Css - stylus
  - Vue.js - Vue cli 3
  - ESlint - Standard Style
  

### 安裝

請先安裝好 Vue cli 3

```sh
$ cd vueSlack
$ npm install
$ npm run serve
```
